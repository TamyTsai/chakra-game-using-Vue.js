- 前端：HTML、CSS、Vue.js
- 部署：GitHub Pages

## 使用技術詳細說明
- 以HTML、CSS及JavaScript撰寫，為一頁式靜態網頁
- CSS中盡量將元素大小以相對單位設定，以達RWD效果
- 使用Vue.js框架
- 採用UJS寫法，維持HTML簡潔
- 運用AJAX設計遊戲機制
